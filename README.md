# Kushal B T - Portfolio

A premium, minimalistic one-page developer portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack
- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Icons:** Iconoir
- **Animations:** Framer Motion
- **Fonts:** Silkscreen (Pixel), Inter (Sans)

## Design Features
- **Theme:** Ultra-dark mode (#0a0a0a)
- **Background:** Subtle Vercel-style grid (5% opacity)
- **Aesthetics:** Glassmorphism, radial glows, and micro-animations
- **Typography:** Dual-font system (Pixelated accents + Modern body)

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Open the browser:**
   Navigate to `http://localhost:3000`

## Project Structure
- `src/app`: Layout and main page
- `src/components`: Modular UI sections (Hero, About, Projects, etc.)
- `src/app/globals.css`: Custom theme and grid background setup
